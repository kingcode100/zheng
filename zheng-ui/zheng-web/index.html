汽车不可缺少的

制动液

又称为 刹车油 - 液压制动系统中使用的流体

制动液是安全驾驶中最重要的部分。刹车油整体的性能影响着整个车的安全，一般来说，刹车油整体使用不超过两年或者不超过40000km，时间一到必须要进行更换，因为长时间的使用刹车油会变质，影响刹车效果。 刹车油的更换是要注意不同品牌不同类型的刹车油不能进行混合，因为配方的不同会造成刹车液的油性指标下降。如果发现刹车油吸附有杂质或者水分时，要考虑是进行过滤还是更换。还有，刹车油也有一定的刻度范围，记得要在这个范围之间，不能过多也不能过少。

防冻液

全称 防冻冷却液 - 一种基于乙二醇的液体，可以加入水中降低冰点，主要用于汽车散热器。

其实不仅仅是在冬天，在夏天也最好使用防冻液。夏天的温度高，如果把防冻液替换成水，水很容易沸腾，导致发动机内炸锅，而防冻液的沸点比水的沸点要高，可以有效防止炸锅这个问题，保护发动机不受损。 冬天温度低，就更要使用防冻液来防止箱内结冰而破裂，从而影响正常的工作。一般情况下防冻液的选择要根据当地的气温情况，选择比当地最低温度低10℃左右的防冻液是较为合适的。在更换防冻液时，不同品牌的防冻液不能进行混合，防冻液也不能与水进行混合。防冻液的使用期限为两年左右，到了一定时间，记得要进行更换。

波箱油

又称 排挡油 - 波箱油是保持排档系统清洁的油类用品，起到润滑延长传动装置寿命的作用

波箱油基本都能使用于目前市场上自动变速箱，事实上每一款自动变速箱的设计都有不同的技术要求，即使是同一型号的变速箱配置在不同的车型，其扭矩、重量、转速、结构等都会不同，因此原厂都有其自己指定的专用波箱油。而不同的波箱油，正是影响换档品质的原因之一。

中国汽车后市场行业调查分析及发展趋势预测

产业现状
“汽车后市场”，指的是汽车售出之后的维修、保养、服务以及所需的汽车零配件、汽车用品和汽车材料的市场。汽车用品的消耗速度与消耗量均首屈一指，现役汽车的保有量直接决定了汽车用品的市场规模。在汽车售前利润不断下滑的同时，“汽车后市场”逐渐为众多品牌所重视。作为汽车市场的新兴行业，汽车后市场相关行业的发展速度越来越快，规模也越来越大。近年来，汽车美容、维修、养护、配件供应等方面的服务更是势如破竹，汽车后市场抢占市场份额的号角已经吹响。2013年，在“微增长”的预期下，我国的汽车市场依旧保持着持续增长的势头。随着私家车的急剧增加与车主以养代修爱车理念的普及，汽车市场逐渐由汽车营销主导转向汽车消费主导，汽车后市场显示出巨大的潜力，正步入一个黄金发展期。汽车后市场蕴藏着巨大的商机，将决定汽车市场的未来。新年伊始，各大车企纷纷发布最新发展战略，从而吹响了抢占2014年“汽车后市场”份额的号角。中国汽车后市场在未来十至十五年时间内，每年将保持两位数以上的高速增长，2015年中国汽车后市场规模有望增至8312亿元，成为全球最大的汽车后市场，2017年汽车后市场的规模将突破10000亿元。

市场容量
中国产业调研网发布的中国汽车后市场行业调查分析及发展趋势预测报告（2016-2022年）认为，2013年汽车产销分别为2211.68万辆和2198.41万辆，同比增长14.76%和13.87%，再次刷新年度产销纪录。综合行业主要企业的预测意见，2014年汽车市场需求量约为2385万辆至2429万辆，增长速度能够达到8%-10%左右。汽车业是我国的支柱产业之一，改革开放以来，国家频繁出台各种促进汽车产业发展的相关政策，汽车维修、保养等配套产业伴随汽车产业而生，汽车产业发展政策为汽车后市场行业的繁荣起到了不可估量的作用。随着汽车市场增长放缓，市场下伸、发展汽车后市场成为车企、经销商新的市场重点。汽车后市场被称为汽车业的黄金产业，它是指汽车销售以后，围绕汽车使用过程的各种服务。
汽车后市场被称为汽车业的黄金产业，它是指汽车销售以后，围绕汽车使用过程的各种服务。汽车后市场大体上可分为七大行业：汽保行业；汽车金融行业；汽车IT行业；汽车精品、用品、美容、快修及改装行业又称汽车养护行业；汽车维修及配件行业；汽车文化及汽车运动行业；二手车及汽车租赁行业。相比发展成熟的国外汽车市场，汽车后市场在国内尚是发展的蓝海。随着国内汽车市场的逐渐成熟，汽车产业链越来越长，中国汽车后市场的高速发展期即将到来。预计未来几年我国汽车售后服务占汽车经销商净利润的比重将逐渐上升，售后市场规模仍将保持20%左右的增速。到2015年国内汽车售后市场规模达到8000亿元以上。

市场格局
巨大的市场必然促进汽车营销模式的重大变化，汽车4S店、汽车城、汽车园区、汽车城市综合体、多品牌汽车超市、网上车市、汽车销售集团、小型2S店、城市展厅、汽车产业园、汽车连锁店、汽车特约维修中心、汽车配件及汽车用品综合市场、专业市场、汽车以租代售等等渠道模式，将会进入到一个多元化的高速竞争时代。总之，渠道多元、渠道升级、渠道网络、渠道合并、渠道下沉、渠道精细化等管理，将成为渠道创新的关键阶段。未来的渠道模式必将由很多能够满足不同区域特点的求道模式共同构成。但永恒不变的法则是高效率最优化。

前景预测
2014年三四线市场有8亿人口已经进入了汽车消费的第一高速增长期。汽车消费增长最快的区域主要分布在中西部较为发达的地级城市、东部富裕的县级城市及辖区。之所以对于未来车市抱有足够的信心，主要是源于三四线市场汽车消费的火爆。与备受限购控制的一二线汽车市场的降温不同，三四线城市在国内新车市场中所占的份额从2012年的26.7%上升到2013年的30%，一线城市的份额则从前年的35.7%降到2013年的30.8%。可以说，随着各大汽车品牌加快渠道下沉速度，三四五线城市占的市场份额会越来越重，并逐渐成为国内汽车营销的主战场。由于现在新的汽车增量主要来自于三四线城市，市场增长点从中心城市向中小城市转移，这是一个很重要的转变。未来由于消费者需求变化的驱动，一方面渠道网络将由中心城市向中小城市渗透，另一方面将推动出现多样化的渠道模式。由于四五线市场的道路建设赶不上经济和汽车的发展，堵车已经不再是大城市的专利。
